## Versions info:

- **PR** *Printing Ready: Scaled 1:1 and places on the right rotation and location.*
- **E_IN_OUT** *Scaled: IN is virtual dim multiplier; OUT is real dim multiplier.*
- **B0000000000** *Building versión where 0000000000 is AA:MM:DD:HH:MM correspondance.*
- **V0.0** *Design versión.*
- **ETA000000** *Estimate time average where 000000 is HHMMSS por an especific machine. (default=Prusa i3, RAMPS 1.4, Direct Driver extruder)*
